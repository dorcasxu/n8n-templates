# 🌟 10个精选 n8n 自动化模板

> 🚀 精选自飞书云文档《DeepSeek 使用指南》系列，整理了 10 个最实用的 n8n 自动化模板，覆盖微信公众号、视频采集、SEO监测、GitHub 热榜等多个高效应用场景。

---

## 🧭 项目简介

本项目是一个 **n8n 自动化工作流模板集锦**，旨在帮助开发者、内容创作者和自动化爱好者更高效地完成各种重复性任务。
所有模板均可直接导入 n8n，但需要根据指引调整参数（如 API Key、Webhook 等）才能正常使用。

## 🧙‍♂️ 如何使用

1.  访问你感兴趣的模板链接。
2.  点击 **"Import into n8n"** 或按教程指引操作。
3.  在你的 n8n 中调整参数（如 API Key、Webhook 等）。
4.  激活工作流，即可开始自动化！
